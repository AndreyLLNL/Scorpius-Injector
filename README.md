# Scorpius-Injector
Model represents Digital Twin of the actual E-Beam Injector to Linear Inductive Accelerator

## Overview

This repository contains a Particle-In-Cell (PIC) simulation of the Pierce Gun electron injector for a Linear Inductive Accelerator. The model includes space charge, magnetic focusing (49 solenoids), multi-pulse operation, ion accumulation, gas scattering, and Schottky-enhanced thermionic emission.

## File Structure

### Simulation Pipeline (Execution Order)

1. **`Pierce_Gun_Parametric_Geometry_Builder_V41.m`** — Builds the Pierce gun geometry and saves it to a `.mat` file. Run this first to generate `pierce_gun_geometry.mat`.

2. **`Pirce_Gun_field_solver_with_capping_updated.m`** — Solves the electrostatic field (Poisson equation) on the Pierce gun geometry and saves the field solution to `pierce_gun_field_solution_capped.mat`. Run after the geometry builder.

3. **`Injector_Model_Main_Loop.m`** — Main entry point for the PIC simulation. Loads geometry and field data, initializes all physics modules, runs the time-stepping loop, and saves the workspace. Run after geometry and field are available.

4. **`Injector_Model_Diagnostics.m`** — Post-processing script. Generates all analysis figures (Figures 1–36+), computes Twiss parameters, performs inter-pulse cloud analysis, and exports results. Run after `Injector_Model_Main_Loop.m` (workspace must be loaded or `simulation_workspace.mat` available).

### Helper Function Files

These are standalone MATLAB function files called by the main simulation:

| File | Description |
|------|-------------|
| `pulse_shape_func.m` | Single-pulse temporal shape (cosine ramp). Called as a function handle in the main loop. |
| `pulse_shape_multipulse.m` | Multi-pulse shape: sums contributions from all configured pulses. Depends on `pulse_shape_func`. |
| `calculate_Bz.m` | Computes the axial magnetic field Bz at a given (z, r) position from all 45 active transport solenoids. Includes `solenoid_Bz` as a subfunction. |
| `calculate_Br.m` | Computes the radial magnetic field Br at a given (z, r) position using the Bz gradient. Includes `solenoid_Bz` and `solenoid_Br` as subfunctions. |

### Legacy / Reference File

| File | Description |
|------|-------------|
| `Pierce_Gun_PIC_v8_Integrated_SC_Extended_V1.m` | Original monolithic 13,600-line script. Preserved for backward compatibility and as reference. |
| `Test_magnetic_field_v2.m` | Stand-alone magnetic field validation test. |

## Dependencies

The simulation requires the following `.mat` data files (generated by the pipeline scripts above):

- `pierce_gun_geometry.mat` — Electrode geometry mesh
- `pierce_gun_field_solution_capped.mat` — Electrostatic field solution

Output files produced by the simulation:

- `simulation_workspace.mat` — Full simulation workspace (saved by main loop)
- `twiss_analysis_results.mat` — Twiss parameter results
- `twiss_betatron_averaged_results.mat` — Betatron-averaged Twiss results
- `interpulse_electron_cloud.mat` — Inter-pulse electron cloud data
- `interpulse_cloud_metrics.mat` — Quantitative cloud metrics
- `exit_particles_*.mat` — Particle handoff files for downstream transport

## Key Configuration Switches

Configured at the top of `Injector_Model_Main_Loop.m`:

| Switch | Values | Description |
|--------|--------|-------------|
| `simulation_mode` | `'QUICK_TEST'`, `'OPTIMIZATION'`, `'PRODUCTION'` | Controls time step, particle count, and grid resolution |
| `ENABLE_SPACE_CHARGE` | `true` / `false` | Enables/disables the Poisson space charge solver |
| `ENABLE_MULTIPULSE` | `true` / `false` | Single-pulse vs. multi-pulse operation |
| `ENABLE_BETATRON_AVERAGING` | `true` / `false` | Multi-snapshot betatron phase averaging for Twiss analysis |
| `ENABLE_ION_ACCUMULATION` | `true` / `false` | Ion creation and accumulation from beam-gas interactions |
| `ENABLE_GAS_SCATTERING` | `true` / `false` | Coulomb and neutral gas scattering |

## Solenoid Configuration

The beamline contains **49 solenoids** (with 4 positions reserved for steering magnets that are off):
- Solenoid 1: Cathode focusing solenoid (z = −279 mm)
- Solenoid 2: Anode solenoid (z = 372 mm)
- Solenoids 3–49: Drift transport solenoids (z = 1144–7448 mm)
- Steering magnet positions 6, 13, 37, 44: Not active in current configuration

Field strengths are set via `solenoidN_field` variables in `Injector_Model_Main_Loop.m`.

## Running the Simulation

```matlab
% Step 1: Build geometry (run once)
run('Pierce_Gun_Parametric_Geometry_Builder_V41.m');

% Step 2: Solve electrostatic field (run once, or when geometry changes)
run('Pirce_Gun_field_solver_with_capping_updated.m');

% Step 3: Run main PIC simulation
run('Injector_Model_Main_Loop.m');

% Step 4: Run post-processing diagnostics
run('Injector_Model_Diagnostics.m');
```

Or, to load a previously saved workspace and re-run only diagnostics:

```matlab
load('simulation_workspace.mat');
run('Injector_Model_Diagnostics.m');
```
